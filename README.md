üåê How to Host Live (GitHub Pages)

You can make this dashboard accessible via a live URL using GitHub Pages:

Ensure your main dashboard file is named index.html.

Go to your repository Settings.

Navigate to the Pages section (on the left sidebar).

Under Build and deployment > Source, select Deploy from a branch.

Under Branch, select main (or master) and / (root).

Click Save.

Wait about 30-60 seconds, and your live link will appear at the top of the page (e.g., https://yourusername.github.io/veralia-dashboard/).
